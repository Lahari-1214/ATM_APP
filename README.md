<h2>Title:</h2>
<h1>ATM Application Using Python Flask Framework</h1>
<h2>Objective:</h2>
<ol>
<li>
The objective of this project is to develop a simple ATM system using the Flask web framework in Python, where users can create accounts, deposit, withdraw, and delete accounts. This application demonstrates basic CRUD operations and memory-based data storage without using a database. It is suitable for beginners to understand form handling, routing, and dictionary-based storage in Flask.</li></ol>
